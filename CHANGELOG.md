# Changelog

## 1.0.0 — 2026-07-17

- Initial production-oriented Oracle Cloud and GitHub deployment package.
- Added phone pairing and QR linking, durable queue, delivery events and signed webhooks.
- Added Terraform provisioning, private versioned state bootstrap and reserved public IP.
- Added Caddy HTTPS, automatic container updates, daily local backups and phone-only documentation.
- Added OCI IMDSv2-backed configuration synchronization and Docker metadata isolation.
- Updated the OCI Terraform provider constraint to the current 8.23.0 release.
- Updated GitHub Actions to the July 2026 supported major lines: Checkout v7, Docker setup/login v4, Build-Push v7 and Setup Terraform v4.
